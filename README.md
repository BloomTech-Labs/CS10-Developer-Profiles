# Stack

### **Frontend**

> - JS Framework: _React_
> - CSS Freamwork: _Material Design_
> - HTTP Client: _Axios_

### **Backend**

> - Framework: _Node_
> - Server: _Express_
> - DB: _Mongo_
> - Encryption: _bcrypt_
> - Deployment platform: _Heroku - mLab_
